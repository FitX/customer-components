## Hausschrift Niveau Grotesk
Alle Komponenten gehen von der Hausschrift Niveau Grotesk aus, diese steht über die css Variable --font-stack-normal bereit.

Die Schrift muss auf Body oder App Root eingebunden werden da es sonst Render Probleme geben könnte.
```css
body {
  font-family: var(--font-stack-normal);
}
```
