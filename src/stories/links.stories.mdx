Keine eigene Komponente. Links werden als [SCSS Styles mit ausgeliefert](src/assets/styles/links.scss).

<Story name="Usage"></Story>

## Beispiel Implementierung in der App
```css
a:any-link {
  @include text-links();
}
```

## Beispiel Implementierung in Komponenten
```css
.link {
  @include text-links();
}

.decent {
  @include text-links-decent();
  .demo--dark & {
    @include text-links-decent(dark);
  }
}
```
